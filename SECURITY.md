# Security Policy

## Reporting a Vulnerability

Please use [GitHub's private vulnerability reporting](https://github.com/EHegyi-Sphericity/pocket-rdf/security/advisories/new) to report security issues.

Please do **not** open a public GitHub issue for security vulnerabilities.

You can expect an initial response within 7 days. Once the issue is confirmed,
a fix will be prioritised and released as soon as practical.
